# Installation instructions
1. Unzip cs-3110-final-project.zip
2. Open cs-3110-final-project in Visual Studio Code
3. Run "dune build" in the terminal
4. Run "make start" in terminal
5. Follow the prompts in the terminal to interact with the VenmOCaml system
